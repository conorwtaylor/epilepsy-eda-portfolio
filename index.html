Exploratory Data Analysis: EEG Seizure Recognition

This project explores an EEG dataset of 11,500 samples using Python to uncover patterns related to epileptic seizure activity. It focuses on identifying meaningful features through data visualization and correlation analysis.

Dataset

  • Source: Kaggle
  • 178 EEG features per sample
  • Class 1: seizure, Classes 2–5: non-seizure

Tools Used

Python (Pandas, Seaborn, Matplotlib)

Visual Examples

Key Insights & Next Steps

  • Clear class imbalance: ~20% of data are seizures
  • Strong variance in features 10–20 for seizure events
  • Heatmap reveals mild-to-moderate correlation in mid-range EEG features
  • Next: Build a classifier (Random Forest or CNN)
